# GrupoJaP3
                                                                                                                                                                                       
# SOLUGINAMA
Soluginama es un sitio de eCommerce donde se puede encontrar una amplia gama de productos, que va desde autos hasta juguetes. 
Este sitio fue desarrollado en el marco del proyecto semestral de la Fase 1 de Jovenes a Programar de Ceibal. 

# Tecnologías utilizadas
En este proyecto se utilizó lo aprendido en clase:
- HTML
- CSS
- JSON
- Bootstrap
- Figma
- NODE.js
- Express.js
- MySQL

# Instructivo paso a paso de como ejecutar el proyecto del backend

1. Abrir en Visual Studio Code los archivos Frontend y Backend.
   - El Backend se encuentran en: https://github.com/Nachoofiore/BackendGrupoJaP
   - El Frontend se encuentra en: https://github.com/Nachoofiore/GrupoJaP4 
2. En el archivo Backend seleccionar la terminal Git Bash e ingresar “npm run dev”.
3. Abrir el Frontend en la web.
4. Ingresar al eCommerce, en este caso “Soluginama”, con uno de los usuarios habilitados.
   - Los usuarios habilitados se encuentran en la carpeta OUTH dentro del Backend.
